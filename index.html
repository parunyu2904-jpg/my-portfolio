Portfolio ด้านเทคโนโลยีสารสนเทศ (IT)

ข้าพเจ้ามีความสนใจในสาขาเทคโนโลยีสารสนเทศจากการได้เรียนรู้และใช้งานคอมพิวเตอร์ในชีวิตประจำวัน ซึ่งทำให้เห็นว่าเทคโนโลยีมีบทบาทสำคัญต่อการพัฒนาสังคมและการแก้ไขปัญหาต่าง ๆ ผมเชื่อมั่นว่าการศึกษาในสาขาไอทีจะเปิดโอกาสให้ผมได้สร้างสรรค์สิ่งใหม่ ๆ และทำให้ชีวิตของผู้คนดีขึ้น

1. ประวัติส่วนตัวและประวัติการศึกษา

ชื่อ: นาย ภรัณยู มยูรกุล

โรงเรียน: บุญวาทย์วิทยาลัย

สายการเรียน: วิทย์–คณิต (แผนการเรียนคอมพิวเตอร์)

2. แรงบันดาลใจและความสนใจด้าน IT

ผมมีความชอบด้านคอมพิวเตอร์และเทคโนโลยีสารสนเทศ จากการเรียนรู้และฝึกฝนด้วยตนเอง ทำให้เกิดความสนใจในการนำเทคโนโลยีมาใช้ แก้ไขปัญหาและพัฒนาสิ่งใหม่ ๆ ที่เป็นประโยชน์ต่อสังคม

3. ทักษะและความเชี่ยวชาญด้านไอที

ทักษะ รายละเอียด ระดับ ตัวอย่างการใช้งาน
ความรู้พื้นฐานคอมพิวเตอร์ เข้าใจการทำงานของฮาร์ดแวร์และซอฟต์แวร์ พื้นฐาน / ปานกลาง ดูแลคอมพิวเตอร์ แก้ปัญหาเบื้องต้น
การเขียนโปรแกรม Python, C, HTML พื้นฐาน / ปานกลาง เขียนโปรแกรมคำนวณ / เว็บเพจ
ซอฟต์แวร์และเครื่องมือ VS Code, Microsoft Office, Google Workspace พื้นฐาน / ปานกลาง ทำรายงานและพัฒนาโปรแกรม
การคิดวิเคราะห์ วิเคราะห์ปัญหาอย่างเป็นขั้นตอน ดี แก้โจทย์โปรแกรมและบั๊ก
การเรียนรู้ด้วยตนเอง ศึกษาจากแหล่งออนไลน์ ดี YouTube, เว็บไซต์, คอร์สออนไลน์

4. ผลงานหรือกิจกรรมที่เกี่ยวข้องกับไอที

  • โครงงานและโปรเจกต์ด้านคอมพิวเตอร์
  • กิจกรรมหรือค่ายด้านเทคโนโลยี
  • การฝึกฝนเขียนโปรแกรมด้วยตนเอง

5. เป้าหมายทางการศึกษา

ที่มหาวิทยาลัยเชียงใหม่ เพื่อพัฒนาความรู้และทักษะด้านซอฟต์แวร์ และนำไปต่อยอดเป็นอาชีพในอนาคต
